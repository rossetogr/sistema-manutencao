# sistema-manutencao
Sistema CLI de Manutenção Preditiva com ML de Regressão. O projeto simula a previsão da Vida Útil Restante (RUL) de equipamentos a partir de leituras de sensores (foco na Vibração). A RUL predita classifica o risco (ALTO/MÉDIO/BAIXO). O MAE (Erro Absoluto Médio) é usado para validar o desempenho do modelo comparando-o com o RUL Real (Ground Truth).
